# Databricks Medallion Retail Analytics

A retail analytics project built using Databricks following the Medallion Architecture (Bronze, Silver, Gold layers).

## Project Workflow
- Bronze Layer: Raw data ingestion
- Silver Layer: Data cleansing and transformations
- Gold Layer: Business-ready analytical models
- Dashboard SQL: Analytical queries for reporting
- Dashboard Visualizations: Interactive business insights

## Technologies Used
- Databricks
- Apache Spark
- PySpark
- SQL
- Delta Lake
- GitHub
